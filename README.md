# Sabor Urbano

Proyecto de ejemplo de menú virtual y landing page para un restaurante de comida rápida.

## Descripción

`Sabor Urbano` es una maqueta moderna diseñada para mostrar un menú virtual atractivo de salchipapas, perros calientes y hamburguesas. Incluye:

- Navegación fija con menú estilo hamburger en móviles.
- Secciones de productos con tarjetas visuales.
- Botones destacados y estilo profesional inspirado en apps de comida rápida.
- Diseño responsive optimizado para móviles y desktop.
- Animaciones suaves al hacer scroll.

## Ideal para

- Presentar un concepto de restaurante de comida rápida.
- Mostrar una landing page de menú virtual a clientes.
- Usarse como base para un portafolio o demo de diseño web.

## Estructura del proyecto

- `index.html` — página principal.
- `css/styles.css` — estilos del sitio.
- `js/script.js` — lógica de menú móvil y animaciones suaves.
- `images/` — recursos visuales y placeholders.

## Cómo usarlo

1. Abre `index.html` directamente en el navegador.
2. O ejecuta un servidor local si prefieres:

```bash
cd "c:\Users\yo\Desktop\landingProyect\sabor-urbano"
python -m http.server 8000
```

3. Abre `http://localhost:8000`.

## Cómo mostrarlo a otras personas

- Sube la carpeta a un repositorio de GitHub y usa GitHub Pages.
- O usa Netlify Drop: arrastra la carpeta `sabor-urbano` a `https://app.netlify.com/drop`.

## Personalización rápida

- Cambia los textos y precios en `index.html`.
- Ajusta colores y fuentes en `css/styles.css`.
- Sustituye las imágenes por fotos reales en `images/`.
